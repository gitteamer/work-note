打开http://localhost:8070

初始用户名是apollo，密码是admin。

### 添加部门
右上角 -> 管理员工具 -> (key = organizations) 查询
```text
key     => organizations
value   => [{"orgId":"TEST1","orgName":"样例部门1"},{"orgId":"TEST2","orgName":"样例部门2"}]
comment => 部门列表
```
修改部门信息后，保存。保存一分钟之后生效。
